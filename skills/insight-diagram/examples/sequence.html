INBOUND

订单入口

  • • RabbitMQ Queue: orderRiskCheck
  • • AMQP TLS 认证
  • • IdempotentStore 去重，跳过重复消息
  • • 手动确认消息，WithoutCancel 保优雅关闭

SCORING

核心风控

  • • GPT Engine: --format=json --model gpt-4-turbo
  • • 3min 独立超时，RiskResult[] 返回
  • • 失败降级 → rule_engine.py (Fallback)
  • • PostProcess: extractJSON → validate → normalize → sort

METADATA

元数据与下载

  • • OrderService: GET /order/:id/detail
  • • 30s 轮询至 success/failed，5min 超时
  • • 数据库导出: dbcli export，5min 超时
  • • DBConfig {region, ak, sk, path}

ALERT

告警发送

  • • POST /job/:jobId/score-result
  • • X-Auth-Token 鉴权，3 次重试
  • • scan_status: 1=诊断中 2=完成 3=失败
  • • 完成/失败均上报，offset 始终提交
