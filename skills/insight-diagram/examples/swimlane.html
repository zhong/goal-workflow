消息消费 → 数据补充 → 风控评分 → 告警发送，4 条泳道展示控制权流转

泳道图 (Swimlane Diagram) — 行为性图形。跨职能流程图，展示不同角色或部门间的职责划分与协作流程。

HANDOFF

跨泳道交接

  • • Ingest → Collect: 去重通过后传递 ScanInput
  • • Enrich → Score: 验证通过后交付特征摘要
  • • Score → Alert: 后处理完成交付 RiskResult[]
  • • 交接用 accent 标记，是最关键的耦合点

FALLBACK

降级路径

  • • GPT 风控评分失败 → rule_engine.py 降级
  • • 降级结果自动走后处理 + 告警
  • • 任何非-alert 阶段失败统一走 fail() 出口
