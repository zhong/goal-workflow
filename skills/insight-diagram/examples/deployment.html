部署图 (Deployment Diagram) — 结构性图形。展示系统的软件如何映射到硬件上，描述物理配置与节点分布。

DEPLOYMENT

单副本部署

  • • nova-shield Go 二进制，部署在 K8s Pod 内
  • • RabbitMQ Subscriber 单实例消费 orderRiskCheck Topic
  • • Healthz :8080 探活，SIGTERM 优雅关闭
  • • 依赖 gpt-cli 和 dbcli 两个 CLI 二进制
  • • IdempotentStore 优先 Redis，失败降级内存

EXTERNAL

外部依赖

  • • RabbitMQ: AMQP TLS 认证
  • • OrderService: X-Auth-Token 鉴权，轮询元数据
  • • CustomerDB: dbcli export 拉取交易数据
  • • GPT Engine: 本地二进制, 5min 超时, 不可用则 Fallback
  • • AlertService: X-Auth-Token, 3 次重试
